# Mnemosyne Evolve operations

## Intent
Use this skill to extract reusable working patterns from repeated signals without polluting durable memory.

## Recommended loop
1. ingest normalized events
2. synthesize candidate patterns
3. review candidate markdown
4. approve a small set of ids
5. build a recall pack before a new task

## Good candidates
- repeated owner preferences
- repeated successful operating sequence
- repeated failure that should be avoided next time
- stable tactic that fits the owner's style

## Bad candidates
- one-off emotional reactions
- noisy transcript fragments
- speculative conclusions without repeated evidence
